# tg-form-filler

Telegram-бот, который принимает сообщение от администратора, извлекает данные с помощью LLM (DeepSeek) и заполняет Google Формы.

## Возможности

- Автоматически определяет нужную форму по тексту сообщения
- Поддерживает несколько форм одновременно (дневник питания, дневник расходов)
- Категория приёма пищи определяется автоматически по текущему времени (МСК), если не указана явно
- Напоминает о записи приёма пищи, если прошло более 8 часов — только в дневные часы (08:00–22:00 МСК)
- Ведёт статистику расходов в памяти и каждое утро в 9:00 МСК отправляет краткий LLM-отчёт о тратах за прошлый день

## Настройка

Скопируй `.env.example` → `.env` и заполни переменные:

```
DEEPSEEK_API_KEY=
TG_BOT_TOKEN=
TG_ADMIN_CHAT_ID=
```

Скопируй `.example` конфиги из директории `form_configs` и заполни их реальными `form_url` и `entry.*` идентификаторами полей.

## Локальный запуск

```bash
docker compose up -d
```

## Деплой на сервер

На сервере нужен только Docker.

Впиши необходимые алиасы в `deploy/deploy.sh` и запусти:

```bash
./deploy/deploy.sh
```
