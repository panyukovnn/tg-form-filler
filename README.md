# tg-form-filler

Telegram-бот, который принимает сообщение от администратора, извлекает данные с помощью LLM и заполняет Google формы.

## Настройка

Скопируй `.env.example` → `.env` и заполни переменные:

```
DEEPSEEK_API_KEY=
TG_BOT_TOKEN=
TG_ADMIN_CHAT_ID=
```

Скопируй `.example` формы из директории `form_configs` и заполни их. 

Данный проект предназначен для заполнения конкретных форм автора, поэтому имеет дополнительную логику, направленную на работы только с ними.

## Локальный запуск

```bash
docker compose up -d
```

## Деплой на сервер

На сервере нужен только Docker.

**Первый раз** — отправить конфиг:
```bash
./deploy/init-server.sh
```

**Обновление кода и перезапуск:**
```bash
./deploy/deploy.sh
```
